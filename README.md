# ChatGPT 对话快捷删除

这是一个 Chrome / Edge 浏览器扩展。打开 ChatGPT 后，页面右上角会出现一个简约总开关。开启后，把鼠标放到左侧对话记录上，会在原本“更多”按钮左侧显示一个垃圾桶按钮。点击垃圾桶后，插件会自动点开该对话自己的菜单，并完成删除确认。

## 安装方式

1. 打开 Chrome 或 Edge 的扩展管理页面。
2. 开启“开发者模式”。
3. 选择“加载已解压的扩展”。
4. 选择这个文件夹：`C:\Users\kexi\Documents\second\chatgpt-delete-helper`

## 使用说明

- 支持 `chatgpt.com` 和 `chat.openai.com`。
- 插件默认关闭，需要先打开页面右上角的总开关。
- 删除按钮默认只在鼠标悬停某条对话时出现。
- 按钮位置放在原“更多”按钮左侧，避免和 ChatGPT 自带菜单重合。
- 删除是直接执行的，不会再出现插件自己的二次确认。
- 自动删除过程中，ChatGPT 自带菜单和确认框会被短暂降到近乎透明，减少闪现感。

## 说明

插件通过模拟页面上的官方删除操作来工作，不保存账号信息，也不需要 API Key。ChatGPT 网页结构更新时，可能需要跟着微调选择器。
